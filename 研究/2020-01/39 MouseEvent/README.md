#### MouseEvent

```
screenX, screenY  鼠标位于用户屏幕的 x, y 坐标
clientX, clientY 鼠标位于客户端窗口的 x, y 坐标
offsetX, offsetY  鼠标位于事件源的 x, y 坐标系
pageX, pageY 鼠标位于整体页面的左上角 x, y 坐标系

ctrlKey     标明是否同时按下 ctrl 键
shiftKey    标明是否同时按下 shift 键
altKey      标明是否同时按下 alt 键
metaKey     标明是否同时按下 meta 键
button      描述了当事件发生时，哪个按键被按下或释放
```
